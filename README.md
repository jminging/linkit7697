# linkit7697

### Usage
```
pio platform install https://github.com/jminging/linkit7697
```


### configuration
```
[env:linkit7697]
platform = linkit7697
board = linkit7697
framework = arduino
monitor_speed = 115200
platform_packages = 
    framework-arduinolinkit7697 @ https://github.com/jminging/framework-arduinolinkit7697.git
    tool-mt76x7-uploader @ https://github.com/jminging/mt76x7-uploader.git
```
